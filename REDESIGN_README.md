# Website Redesign - Minimalistic Version

## Overview
This is a minimalistic redesign of Eduardo M. Gutiérrez's academic website. The redesign focuses on clean typography, improved readability, and modern web standards.

## Key Changes

### Design Philosophy
- **Minimalistic approach**: Removed decorative elements in favor of clean, functional design
- **Typography-focused**: Used Inter font family for improved readability
- **White space**: Generous use of white space to improve content hierarchy
- **Mobile-first**: Responsive design that works well on all devices

### Technical Improvements
- **Modern CSS**: Uses CSS Grid and Flexbox for layout
- **Performance**: Optimized images and reduced CSS complexity
- **Accessibility**: Better color contrast and keyboard navigation
- **SEO**: Improved semantic HTML structure

### Visual Changes
- **Color scheme**: Simplified to primarily black, white, and blue accent
- **Navigation**: Fixed navigation bar with smooth scrolling
- **Layout**: Single-page layout with smooth section transitions
- **Cards**: Research projects displayed in clean card layout
- **Buttons**: Modern button styling with hover effects

### Features Added
- **Dark mode support**: Automatic dark mode based on user preference
- **Mobile menu**: Hamburger menu for mobile devices
- **Smooth scrolling**: Better user experience when navigating
- **Active navigation**: Highlights current section in navigation

## Files

### New Files
- `assets/css/minimal.css` - New minimalistic stylesheet
- `index-minimal.html` - Clean HTML structure
- `index-original.html` - Backup of original design

### Original Files Preserved
- All original assets and images remain unchanged
- Original `index.html` is backed up as `index-original.html`
- Original CSS files remain for reference

## Browser Support
- Modern browsers (Chrome, Firefox, Safari, Edge)
- Mobile browsers (iOS Safari, Chrome Mobile)
- Graceful degradation for older browsers
